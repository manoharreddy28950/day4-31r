INTRODUCTION TO VSCODE

  1. Lightweight and Powerful: VS Code is a fast, cross-platform code editor developed by Microsoft.
  2. Rich Features: Includes IntelliSense, debugging, and Git integration for seamless development.
  3. Highly Extensible: Supports extensions for languages, themes, and tools to enhance functionality.
  4. User-Friendly Interface: Offers features like syntax highlighting, auto-completion, and split views.
  5. Cross-Platform: Available for Windows, macOS, and Linux, making it accessible to a wide range of users.

FEATURES OF VS CODE

  1. IntelliSense: Offers smart code completions, parameter hints, and syntax highlighting for various programming languages.
  2. Built-in Debugger: Provides debugging tools for multiple languages, including breakpoints, watch variables, and a debug console.
  3. Extensions Marketplace: Access thousands of extensions for additional languages, themes, and development tools.
  4. Integrated Git: Includes version control features like staging, committing, and viewing Git changes directly within the editor.
  5. Customizable UI: Allows users to tweak themes, layouts, and keyboard shortcuts to match their workflow.

EMMET SYMBOLS

EMMET SYMBOL DESCRIPTION
> Used to create child elements within a parent element
+ Used to create siblings
^ Moves up one level in the hierarchy
* Creates multiple elements
{ } Used for content
[ ] Used for attributes

SHORTCUT KEYS

KEYS DESCRIPTION
Ctrl + S To save the current file
Ctrl + O To open the selected file
Ctrl +C To copy the selected text
Ctrl + V To paste the selected text
Ctrl +X To cut the text
Ctrl +Z To return the previous text
Ctrl + A To select all the content file
Ctrl + Shift + P To open command palette
Ctrl + Shift +N To open new window
